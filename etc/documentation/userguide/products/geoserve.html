Types

geoserve
An aggregate product composed of information fetched from the GeoServe web service.

Properties

utcOffset
[Integer] Minutes between epicenter timezone and UTC.
tsunamiFlag
[“true”|“false”] Boolean string indicating if the tsunami flag should be triggered automatically.
location
The location name for the earthquake. Event page title.

Content

geoserve.json
[application/json] Raw JSON string of the response from the GeoServe web service. Note: There are may properties on this JSON object that are not sent as part of the official product. Generally they are split off into their component properties and sent separately.
